# 🎬 Movie Recommender System

A Netflix-style movie recommendation system using Machine Learning.

## 🚀 Live App
👉 https://movie-recommender-isha.streamlit.app/

## 📌 Features
- Content-based recommendation
- TMDB API for posters
- Clean UI (Netflix-style)
- Fast similarity search

## 🛠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
